  <files>

  <action>
    1. Run `npx create-next-app@latest . --typescript --tailwind --app --no-src-dir --import-alias "@/*"` in the shrike project directory (C:/Users/steph/OneDrive/Desktop/claude/shrike). If the directory has existing files (.planning), run from a temp dir and move files, or use --yes flag. The key is to get a working Next.js 16 project.

    2. Verify versions: `next@latest` should be 16.1.3+, `react@latest` should be 19.2.4+ (security patches). If not, upgrade explicitly.

    3. Create `lib/fonts.ts` with:
       - Inter from next/font/google (weights 400, 500, 600, 700, variable --font-inter) for body text
       - Geist from next/font/google (if available) OR from next/font/local with woff2 files. Geist is Vercel's font and may be included by default in create-next-app. Use variable --font-geist for headlines.
       - Source_Code_Pro from next/font/google (weights 400, 600, variable --font-mono) for monospace
       - All with subsets: ['latin'], display: 'swap'

    4. Create `app/globals.css` with Tailwind v4 setup:
       ```css
       @import "tailwindcss";

       @custom-variant dark (&:where(.dark, .dark *));

       @theme {
         --font-family-sans: var(--font-inter), system-ui, sans-serif;
         --font-family-display: var(--font-geist), var(--font-inter), system-ui, sans-serif;
         --font-family-mono: var(--font-mono), 'Courier New', monospace;

         --color-bg-primary: oklch(0.13 0.02 260);
         --color-bg-secondary: oklch(0.17 0.02 260);
         --color-bg-tertiary: oklch(0.22 0.02 260);
         --color-text-primary: oklch(0.95 0.01 260);
         --color-text-secondary: oklch(0.70 0.01 260);
         --color-accent: oklch(0.75 0.15 55);
         --color-accent-hover: oklch(0.82 0.15 55);
       }

       /* Reduced motion: disable all animations and transitions */
       @media (prefers-reduced-motion: reduce) {
         *, *::before, *::after {
           animation-duration: 0.01ms !important;
           animation-iteration-count: 1 !important;
           transition-duration: 0.01ms !important;
         }
       }
       ```
       The accent color is a warm gold/amber for cinematic feel. Adjust oklch values if needed for contrast.

    5. Create `app/layout.tsx` as root layout:
       - Import fonts from lib/fonts.ts, apply CSS variable classes to `<html>`
       - Set `<html lang="en" className="dark {fontVars}">`
       - Add inline script in `<head>` to prevent FOUC (dark mode toggle from localStorage)
       - `<body className="bg-bg-primary text-text-primary font-sans antialiased min-h-screen">`
       - Import and render Navigation and Footer components
       - Use semantic HTML: `<header>` (nav), `<main>` (children), `<footer>`
       - Add skip-to-content link as first element: `<a href="#main-content" className="sr-only focus:not-sr-only ...">Skip to content</a>`

    6. Create `app/page.tsx` as homepage placeholder:
       - Simple hero section with "Shrike Media" heading (font-display), tagline
       - Responsive: text sizes scale with breakpoints (text-4xl md:text-6xl lg:text-7xl)
       - Use semantic `<section>` elements

    7. Create `app/not-found.tsx` with dark themed 404 page.

    8. Create `types/portfolio.ts` with Project type:
       ```typescript
       export type ProjectCategory = 'photography' | 'videography' | 'software'

       export interface Project {
         slug: string
         title: string
         description: string
         category: ProjectCategory
         thumbnail: string
         blurDataURL?: string
         videoUrl?: string
         tags: string[]
         publishedAt: string
       }
       ```

    9. Ensure `next.config.ts` is minimal -- no custom webpack config (Turbopack is default in Next.js 16). Add image remotePatterns if needed later. Disable React Compiler for Phase 1 stability:
       ```typescript
       import type { NextConfig } from 'next'
       const nextConfig: NextConfig = {}
       export default nextConfig
       ```

    IMPORTANT: Since this is always-dark (cinematic site), we do NOT need a theme toggle. Hardcode `className="dark"` on html. The localStorage script is a safety net only.
  </action>
  <verify>

  </verify>
